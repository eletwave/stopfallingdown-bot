Converti in token Instagram a lunga durata

Incolla il token breve ottenuto da Instagram e la chiave segreta della nuova app Meta. Il browser invierà i dati direttamente a graph.instagram.com.

La risposta JSON conterrà un nuovo access_token. Quello è il valore da salvare nel GitHub Secret IG_ACCESS_TOKEN.

Il user_id ottenuto nel passaggio precedente va salvato come IG_USER_ID.

Non pubblicare token o chiavi segrete in issue, commit o screenshot.
